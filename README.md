# C++20 Playground

## Features

- Use `conan` to manage dependencies
- Use `Catch2` for testing
- Modern C++ multithreading: `execution::par` (17), `jthread` (20), `latch` (20), `async` (11), `promise` (11), `packaged_tasks` (11)
